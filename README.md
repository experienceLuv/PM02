🚀 Установка и запуск
1. База данных
Установите MS SQL Server

Выполните скрипт scriptDB.sql для создания БД ProductionDB

Выполните скрипты для добавления недостающих таблиц и данных

2. API
Откройте проект ProductionApi в Visual Studio

Проверьте строку подключения в appsettings.json

Нажмите F5 для запуска

Откройте https://localhost:7282/swagger для проверки

3. Клиентские модули
Откройте нужный проект (TechnologistWpf, LabClient, OperatorClient)

Проверьте адрес API в App.xaml.cs

Нажмите F5 для запуска
